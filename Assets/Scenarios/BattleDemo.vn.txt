# ============================================================
# BattleDemo.vn.txt —— 回合制战斗模块演示（事件接口 P4）
# 示范：① 固定数值战斗 ② 养成属性联动（patkstat:体力）③ 车轮战（战斗剩余HP）
# 播放方式：Scenario Editor 打开本文件 → 选中首行 → ▶ 从选中行播放；
#           或把 VNScriptRunner 的剧本换成本文件。
# 战斗操作：点按钮或 1 攻击 / 2 重击 / 3 防御 / 4 逃跑
# ============================================================

: （战斗演示开始。先给主角一点基础属性。）
stat 体力 8
flag 战意 0

# ---------- ① 固定数值的入门战 ----------

label 首战
: （夜路上，一团紫色的影子拦住了去路……）
event battle enemy:暗影史莱姆 ehp:22 eatk:4 php:30 patk:6 escape:60
* 胜利 -> 首战胜利
* 失败 -> 首战失败
* 逃跑 -> 首战逃跑

label 首战胜利
flag 战意 +1
: （击退了史莱姆！信心大增。）
jump 修炼

label 首战失败
: （被史莱姆糊了一脸……先撤退修炼吧。）
jump 修炼

label 首战逃跑
: （三十六计走为上！）
jump 修炼

# ---------- ② 养成联动：体力越高攻击越强 ----------

label 修炼
: （要不要特训一下再挑战强敌？体力会直接变成攻击力。）
choice
* 特训！（体力+6） -> 特训
* 就这样上 -> 强敌战

label 特训
stat 体力 +6
: （挥汗如雨……体力上来了！）
jump 强敌战

label 强敌战
: （森林深处，魔像苏醒了。这次的攻击力 = 当前体力！）
event battle enemy:苔石魔像 ehp:40 eatk:6 php:34 patkstat:体力 pdef:1 escape:0
* 胜利 -> 强敌胜利
* 失败 -> 强敌失败

label 强敌失败
: （魔像太硬了……特训之后再来吧。演示结束。）
jump 收尾

# ---------- ③ 车轮战：剩余 HP 直接带进下一场 ----------

label 强敌胜利
: （魔像轰然倒地！但是——魔像背后还藏着一只小蝙蝠！）
if 战斗剩余HP<=10 jump 重伤劝退
: （乘胜追击！剩余体力直接带入下一战。）
event battle enemy:洞窟蝙蝠 ehp:14 eatk:3 phpstat:战斗剩余HP patkstat:体力 escape:70
* 胜利 -> 连战全胜
* 失败 -> 连战落败
* 逃跑 -> 连战撤退

label 重伤劝退
: （伤得太重，蝙蝠见状识趣地飞走了。见好就收。）
jump 收尾

label 连战全胜
flag 战意 +1
: （车轮战全胜！战意高涨！）
jump 收尾

label 连战落败
: （强弩之末，栽在了小蝙蝠手里……）
jump 收尾

label 连战撤退
: （体力见底，果断撤退。）
jump 收尾

label 收尾
if 战意>=2 jump 完美收官
: （战斗演示结束。属性联动 / 结果分支 / 车轮战都看过了。）
jump 结束

label 完美收官
: （两战全胜收官！战意 flag 也可以像这样跨战斗累计。）

label 结束
: （BattleDemo 播放完毕。）
