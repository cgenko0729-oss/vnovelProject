# ============================================
# VN 剧本演示（P0+P1）— 直接编辑本文件后重新 Play 即可
# 语法速查：
#   bg <背景id> [transition:转场名]
#   show <角色> [at:left|center|right] [expr:表情] [with:出场预设]
#   hide <角色> [with:dissolve|fade]
#   emote <角色> <Surprise|Angry|Shy|Dejected|Recover|Nod|HeadShake>
#   角色 [表情]: 台词        /  旁白: 台词  /  : 无名牌旁白
#   wait <秒> | shake <light|medium|heavy> | sakura
#   camera <pushin|snapzoom|pan|dolly|reset> [参数] [focus:角色]
#   weather <Petals|Rain|Snow|Fireflies|None> | mood <Sunset|Night|...>
#   fx <godrays|dof|clouds|haze|shimmer|heartbeat|dutch> <on|off>
#   行尾加 @ = 不等待该演出完成（异步）
# ---- P1 分支 ----
#   label <名字> / jump <名字>
#   jump <文件>::<名字>            跨文件直达 label（目标文件需登记进 Chapters）
#   call <文件>::<名字> / return    调用公共片段并自动回到 call 的下一行（支持嵌套）
#   params 必填名 可选名=默认值       紧跟目标 label；call 用 名字:值 传参，${名字} 读取
#   flag <名字> [数值|+1|-1|rand:1-100]   rand = 区间内随机取整（含两端）
#   if <表达式> jump <名字>     支持空格、!、整数算术/比较、&&、||、括号
#   例：if 月份 == 12 && (好感度 + 友情度) / 2 >= 50 jump 圣诞约会
#   choice                       下一行起用 * 列出选项：
#   * 选项文本 [flag:名字+1] [-> 标签]   （无 -> = 顺序继续）
# ---- 事件（玩法接口）----
#   event <模块id> [key:value…]      调起 VNEventRegistry 登记的玩法模块
#   * 结果名 [flag:名字+1] [-> 标签]   按模块返回的结果分支（同 choice 写法）
#                                     整数结果会同时写入 flag「事件结果」
# ============================================

bg bg1
fx godrays on
weather Petals
#letterbox on
#fx speedlines on
fx meteor on
mood Dream
fx kenburns on
show 亚里沙 at:left with:DissolveGlow
wait 0.4
show 小雪 at:right with:FadeSlideUp

亚里沙: xxx今天的晚霞真漂亮啊……整片天空都烧起来了一样。
小雪: 是啊。你看，连云的边缘都镶上了金色。

quest start 告白大作战       # 接取 → Toast「新任务：告白大作战」

camera pushin 1.05 5 focus:亚里沙 @
亚里沙: 那个……小雪。
emote 小雪 Surprise
小雪: 怎、怎么了？突然这么严肃。

fx heartbeat on
亚里沙: 我一直……有件事想告诉你。

choice
* 鼓起勇气说出来 -> 告白线
* 还是算了…… -> 退缩线

label 告白线
camera snapzoom 1.1 focus:亚里沙 @
event qte time:3 target:12 title:鼓起勇气连打！
* success flag:好感度+2 -> 告白成功
* fail -> 退缩线

label 告白成功
sakura
亚里沙: 小雪……我喜欢你！从很久以前开始就是！
emote 小雪 Surprise
wait 0.5
emote 小雪 Shy
小雪: ……笨蛋。怎么突然、说这种话啦。
jump 夜晚

label 退缩线
fx heartbeat off
emote 亚里沙 Dejected
亚里沙: ……没、没什么！忘了吧！
emote 小雪 HeadShake
小雪: 什么嘛，真是的。
emote 亚里沙 Recover
jump 夜晚

label 夜晚
fx heartbeat off
camera reset @
bg bg2 transition:Eyelid
mood Night
weather Fireflies
fx godrays off
旁白: ——那天夜里，萤火虫漫天飞舞。

if 好感度 >= 2 && !坏结局锁定 jump 好结局
亚里沙: （总有一天，我一定会说出口的。）
jump 落幕

label 好结局
亚里沙: （今天……是最棒的一天。）
小雪: （明天也想……和她一起看晚霞。）

label 落幕
call 通用送礼 角色:小雪 道具:蝴蝶结 好感:10
hide 小雪 with:fade
hide 亚里沙 with:dissolve
: 第一章　完
jump 演示结束

se s01
