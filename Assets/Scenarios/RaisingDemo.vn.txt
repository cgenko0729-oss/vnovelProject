: （九月，新学期。养成演示开始——先初始化属性与日程。）
stat 金钱 500
stat 行动力 10
stat 压力 0
time set 9 remain:3

label 月初
: （新的一个月开始了，行动力回满。今天做点什么？）

label 行动
if 剩余月数<=0 jump 结局
if 行动力<=0 jump 月末
choice
* 去打工 cost:行动力-2 -> 打工
* 用功学习 cost:行动力-2 -> 学习
* 出门逛服装店 cost:行动力-1 -> 逛街
* 咬牙特训 if:压力<80 cost:行动力-3 -> 特训
* 这个月就到这里 -> 月末

label 打工
stat 金钱 +150
stat 压力 +10
: （在便利店打了一天工，累但充实。）
jump 行动

label 学习
stat 智力 +15
stat 压力 +8
: （在图书馆用功到深夜。）
jump 行动

label 特训
stat 体力 +20
stat 魅力 +10
stat 压力 +20
: （高强度特训！感觉快要到极限了……）
jump 行动

label 逛街
event shop id:服装店
if 道具_蝴蝶结发饰>=1 jump 逛街_有发饰
: （在店里转了一圈就出来了。）
jump 行动
label 逛街_有发饰
: （戴上可爱的发饰，心情变好了！）
stat 压力 -10
jump 行动

label 月末
: （这个月的行动力用完了。）
time pass
jump 月初

label 结局
: （演示期结束！来看看你的养成成果——按 C 打开属性面板。）
if 智力>=40 jump 学霸结局
if 金钱>=800 jump 富豪结局
: （普通结局：平凡而快乐的日子。）
jump 完
label 学霸结局
: （学霸结局：智力大幅成长！）
jump 完
label 富豪结局
: （富豪结局：攒下了一大笔钱！）
label 完
: （养成玩法演示到此结束，感谢游玩！）
